SuperF4
Por Stefan Sundin
Traducido por Fabrizio Ferri (algernon@gmail.com)

Presiona Control+Alt+F4 para terminar el proceso de la ventana seleccionada.
El efecto es el mismo que si mataras el proceso desde el Administrador de tareas.
Esto resulta muy útil en juegos a pantalla completa, y generalmente evita tener que reiniciar.

También puedes terminar un proceso pulsando la tecla Windows+F4 y luego pulsando la ventana con el ratón.
Pulsa Escape o el botón derecho para salir del modo sin terminar ningún programa.
Si matas a explorer.exe accidentalmente puedes volver a arrancarlo desde el Administrador de Tareas (Ctrl+Alt+Esc).
SuperF4 buscará automáticamente las actualizaciones. Puedes desactivar esto en el fichero SuperF4.ini.

Visita la página web en http://code.google.com/p/superf4/
Visita la página web del autor en http://stefansundin.com/

Historial de cambios
--------------------
1.2 - 2010-10-23:
Se añadió un ejecutable de 64-bits, el programa de instalación seleccionará el correcto.
Se añadió un modo de detección de Ctrl+Alt+F4 alternativo (ver SuperF4.ini). Actívalo para que SuperF4 funcione con StarCraft II por ejemplo.
Se añadió xkill al menú de la bandeja del sistema (Win+F4).
Se añadió una opción para comprobación de actualizaciones manual al menú de la bandeja del sistema.
Se arregló el error Shell_NotifyIcon que aparecía para algunas personas al iniciar SuperF4 con el sitema.

1.1 - 2009-08-19:
Arreglado fallo del puntero del ratón que no cambiaba al pulsar Win+F4 con Aero desactivado en Vista.
El cursor del ratón puede moverse ahora entre múltiples pantallas.
El idioma puede cambiarse en el fichero SuperF4.ini.
Corrección de errores.

1.0 - 2009-01-12:
Pide SeDebugPrivilege de forma más segura.
Fundido hooks.dll en SuperF4.exe.
El cursor del ratón cambia al pulsar Windows+F4.
Eliminado fichero log.
Añadida comprobación de actualizaciones.

0.9 - 2008-10-12:
Añadido atajo Windows+F4
Renombrado keyhook.dll a hooks.dll.
Renombrado el fichero log a superf4-log.txt.

0.8 - 2008-09-16:
La detección del atajo Control+Alt+F4 es ahora más segura, evitando terminaciones accidentales que ocurrían cuando SuperF4 no recibía el estado de tecla levantada para F4.
Sólo una instancia de SuperF4 puede ser ejecutada. Abrir el programa otra vez hará reaparecer el icono de tarea si estaba oculto.

0.7 - 2008-05-11:
keyhook intenta ahora obtener el privilegio SeDebugPrivilege, permitiendo la finalización de más programas.

0.6 - 2008-03-15:
Arreglado un pequeño error para evitar que la tecla pulsada se propagase a otra ventana tras terminar el proceso.

0.5 - 2008-03-05:
Arreglado un extraño caso en el que SuperF4 terminaría un procese cuando el usuario pulsara Alt+F4

0.4 - 2008-02-24:
Opción para iniciar SuperF4 con Windows. Se puede ocultar el icono durante la sesión.

0.3 - 2008-02-14:
El icono de notificación puede ocultarse ahora usando el parámetro -hide.

0.2 - 2008-02-08:
El icono de notificación vuelve ahora a añadirse cuando se reinicia Explorer.

0.1 - 2008-02-08:
Primera versión.
